# 1. Análisis de Requisitos

El cliente necesita el despliegue de una infraestructura de Internet basada en la pila LAMP. Esta infraestructura está compuesta por las siguientes herramientas:
* **Linux:** Sistema Operativo base (utilizaremos Ubuntu Server).
* **Apache:** Servidor Web para la página principal.
* **MySQL:** Sistema Gestor de Bases de Datos para la web y gestión interna.
* **PHP:** Lenguaje de programación para dotar de dinamismo a las aplicaciones.
